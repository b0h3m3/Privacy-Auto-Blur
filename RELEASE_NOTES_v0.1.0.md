# Privacy Auto Blur v0.1.0

## Added
- Saved personal-info entries in extension settings
- Auto-detection of matching text on web pages
- Automatic masking for dynamic page updates
- Optional masking for `input` and `textarea` values
- Strong blur plus solid box masking for safer hiding
- Faster partial rescans for improved responsiveness

## Install
1. Download `privacy-auto-blur-v0.1.0.zip`
2. Extract it locally
3. Open `chrome://extensions`
4. Enable Developer Mode
5. Click `Load unpacked`
6. Select the extracted folder

## Notes
- This release uses string matching based on saved entries
- Images, canvas text, and other non-DOM text are not detected yet
