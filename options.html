Chrome Extension

개인정보 자동 블러 설정

이름, 전화번호, 이메일, 주소처럼 페이지에 보이면 가리고 싶은 정보를 저장해 두세요. 텍스트가 포함된 요소를 자동으로 감지해 블러 처리합니다.

현재 버전에서 되는 것

  • 페이지 텍스트에 개인정보가 포함되면 해당 요소를 블러 처리
  • SPA처럼 화면이 동적으로 바뀌는 사이트도 MutationObserver로 재감지
  • 입력창 값도 옵션으로 감지 가능

주의할 점

  • 이미지 안에 박힌 글자나 캔버스에 그려진 텍스트는 감지하지 못함
  • 아주 큰 페이지에서는 감지를 위해 약간의 성능 비용이 생길 수 있음
